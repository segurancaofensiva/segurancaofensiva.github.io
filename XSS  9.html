Cross-site scripting (XSS) 9

Uma técnica de ataque ao navegador do usuário2

É um tipo de vulnerabilidade que permite a injeção de códigos maliciosos em uma aplicação/site web. Esses códigos maliciosos são interpretados/executados pelo navegador do usuário, esse tipo de falha tem como foco o usuário e não o servidor do site(client-side-script). OBS.: Em alguns casos específicos o servidor pode ser afetado pelas operações do lado do usuário. Mas não entraremos em detalhes sobre esse assunto.

Não persistente:
É fornecido uma entrada de dados para o sistema. O sistema irá processar essa entrada de dados e exibir um resultado. Esse resultado é mostrado junto com o código da entrada de dados, se não for tratado, a entrada de dados é processada como sendo parte do código(HTML, javascript).

Passo-a-passo:
1 - Entrar no site(fictício): http://www.buscador.com.br
2 - No campo de busca efetuar a seguinte busca, postando um código javascript: alert("Eu sou um alerta!");
3 - Note que o site irá processar a busca e mostrar o seguinte resultado: Nenhum resultado encontrado para: alert("Eu sou um alerta!");
4 - Se esse site não tivesse tratamento adequado o java script seria executado.
5 - O cracker, poderia distribuir a URL(www.buscador.com.br/busca/Pesquisar&q=alert("Hello!+I+am+an+alert+box!!") gerada pelo site via email por exemplo e todo usuário que abrisse a URL executaria o código achando que é seguro por estar em um site conhecido.

Persistente
A diferença entre o tipo anterior é que a entrada de dados fornecida aqui será armazenada no site.

Passo-a-passo:
1 - O cracker faz um post em um fórum
2 - O contéudo desse post será um código javascript: alert("Eu sou um alerta!");
3 - Os usuário que acessarem aquele post em especifico terão o código javascript executado em seus navegadores.

Outro exemplo(persistente):
1 - Em um sistema de controle de estoque web
2 - Um cracker poderia cadastrar um material com o nome de “alert("Eu sou um alerta!");”.
3 - Sempre que algum usuário for acessar o cadastro de materiais, terá o código executado em seu navegador.
